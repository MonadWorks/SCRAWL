A Chrome extension that captures and organizes your text inputs across the web.

## Features

- Auto-capture text inputs with privacy filtering
- Star and tag important records
- Search and filter by domain
- Local-first storage with IndexedDB
- Export data as JSON

## Install

```bash
npm install
npm run build
```

Load `dist/` folder as unpacked extension in Chrome.
